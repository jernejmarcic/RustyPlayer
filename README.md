# Rustyplayer
 A dead simple (still barely functional) music player written in Rust

## Features
- [x] Plays music
- [x] Pause
- [x] Next
- [x] Shuffle
- [ ] Previous song (the foundaton exists)
- [x] Lopping
- [ ] mpris support
- [x] Album art
